# 聴牌チャンス問題集

エンスクドリルのUI方針をベースにした、聴牌チャンス問題の練習アプリです。

14枚の牌姿から、次に聴牌へ進める有効牌が最も多くなる打牌を選びます。

## 初期仕様

- 問題牌姿の牌を直接タップして回答します。
- 複数正解に対応します。
- 受け入れは残り枚数込みで判定します。
- 七対子を含めて判定します。
- 赤ドラは区別しません。
- 国士無双は初期版では除外します。
- 同じ最大受け入れの打牌が複数ある場合は複数正解です。

## 牌画像

牌画像はエンスク承認済みの `tiles/` 画像を使用します。

## 公開

GitHub Pages で `main` branch の root を公開すれば、そのまま動作します。
